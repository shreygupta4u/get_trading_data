──────────────────────────────────────────────────────────────────
PROJECT OVERVIEW
──────────────────────────────────────────────────────────────────
A Python tool that:
1. Downloads the full list of NASDAQ-listed equities from the official NASDAQ
   FTP symbol directory (pipe-delimited .txt file).
2. Fetches the daily OHLCV candle for each symbol using yfinance's batch
   download API.
3. Stores the results locally as date-partitioned Parquet files
   (Hive-style: data/year=YYYY/month=MM/YYYYMMDD.parquet).
4. Runs on a recurring daily schedule at 4:15 PM local time via APScheduler.
5. On every startup performs an automatic catch-up: back-fills up to 7
   calendar days of missing trading-day data and, if past 4:15 PM and today's
   file is missing, fetches immediately.

### fetch_nasdaq_candles.py
Core fetching and storage logic with:
- setup_logging() → RotatingFileHandler to logs/fetch.log + StreamHandler
- fetch_nasdaq_symbols() → list[str]  — fetches, parses, and cleans the NASDAQ
  symbol file; strips symbols containing $, ^, +, .
- _fetch_batch(symbols, target_date) → pd.DataFrame  — calls yf.download with
  group_by="ticker", auto_adjust=True; flattens MultiIndex to long format with
  columns: symbol, date, Open, High, Low, Close, Volume
- fetch_candles_for_date(symbols, target_date) → pd.DataFrame  — iterates
  batches with retry logic; logs progress every 10 batches
- parquet_path(target_date) → Path  — returns Hive-partitioned path
- save_to_parquet(df, target_date) → Path  — writes parquet, casts date to
  datetime, symbol to category
- record_run(date) / last_run_date() → state file helpers
- is_trading_day(date) → bool  — returns True Mon–Fri (no holiday check)
- run_fetch(target_date, force=False) → bool  — orchestrates one day's fetch;
  skips if file exists unless force=True; warns if run before 4 PM
- run_today(force=False) → bool
- CLI via argparse:
    python fetch_nasdaq_candles.py                          # today
    python fetch_nasdaq_candles.py --date YYYY-MM-DD        # one day
    python fetch_nasdaq_candles.py --date START --date END  # inclusive range
    python fetch_nasdaq_candles.py --force                  # overwrite existing

### scheduler.py
APScheduler (BlockingScheduler) wrapper with:
- Own RotatingFileHandler to logs/scheduler.log
- run_catchup() — finds missing trading days in the past CATCHUP_DAYS, fetches them
- maybe_run_today() — if past schedule time and today's file missing, fetch now
- scheduled_job() — called by the cron trigger
- CronTrigger(hour=SCHEDULE_HOUR, minute=SCHEDULE_MINUTE), max_instances=1,
  coalesce=True, misfire_grace_time=3600
- CLI flags: --catchup-only (run catch-up then exit), --run-now (fetch today then schedule)
- Startup order: run_catchup() → maybe_run_today() → start scheduler
